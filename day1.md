## For Windows users

The current version of ROOT is not supported on Windows, and Unix/Linux is the default in HEP, so we strongly encourage you to get an Unix/Linux installation. Ubuntu is a user-friendly place to start if you've never seen Linux before, and there are ways to install it and keep your Windows install (dual boot or virtual machine).
